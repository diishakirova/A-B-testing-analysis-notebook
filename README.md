# A/B-тестирование и анализ результатов

Аналитический проект по разработке и оценке результатов A/B-теста продуктовой функции.

Проект выполнен в рамках  аналитике данных и направлен на:
- проверку продуктовых гипотез
- анализ поведения пользователей
- оценку влияния изменений на метрики продукта

---

##  Цель проекта

Рассчитать параметры A/B-теста, проверить корректность проведения эксперимента и проанализировать его результаты.

---

##  Данные

Использованы данные о пользовательских сессиях:

- история активности пользователей
- данные тестовой группы
- данные контрольной группы

Основные признаки:
- user_id
- session_id
- session_date
- region
- device
- registration_flag
- page_counter
- test_group

---

##  Используемые инструменты

- Python  
- pandas  
- numpy  
- scipy  
- matplotlib  
- Jupyter Notebook  

---

## Этапы анализа

1. Загрузка данных  
2. Предобработка  
3. Проверка корректности A/B-теста  
4. Расчёт метрик  
5. Статистическая проверка гипотез  
6. Интерпретация результатов  

---

## Результаты

- проведена проверка корректности эксперимента  
- рассчитаны ключевые метрики  
- выполнены статистические тесты  
- сделаны выводы о влиянии изменений на продукт
